# Ghostty

## Font

This config uses [DM Mono Nerd Font](https://github.com/minhuw/dm-mono-nerd-font).

### Install

```sh
git clone --depth 1 https://github.com/minhuw/dm-mono-nerd-font.git /tmp/dm-mono-nerd-font
cp /tmp/dm-mono-nerd-font/dm-mono-nerd-font/*.ttf ~/Library/Fonts/
rm -rf /tmp/dm-mono-nerd-font
```

Then reload Ghostty (`⌘,` → reload config, or restart).

The family name referenced in `config` is `DMMono Nerd Font` (no space between "DM" and "Mono").
